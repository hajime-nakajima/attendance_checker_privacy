Privacy Policy(プライバシーポリシー)

This Privacy Policy describes how this Microsoft Edge extension handles user data.

本プライバシーポリシーは、本 Microsoft Edge 拡張機能における ユーザーデータの取り扱いについて説明するものです。

Scope of Operation(動作対象)

This extension runs only on specific attendance management web pages under the domain p.ieyasu.co.

本拡張機能は、p.ieyasu.co ドメイン配下の特定の勤怠管理ページでのみ動作します。

Data Access and Usage（データの取得および利用）

When a user accesses the supported pages, the extension displays a button. Upon user interaction, the extension reads the working time information currently displayed on the page in order to check for potential input mistakes based on predefined rules.

対象ページにアクセスした際、本拡張機能はボタンを表示します。 ユーザーがボタンをクリックした場合、現在表示されている勤務時間の情報を ブラウザ内部で読み取り、あらかじめ定められたルールに基づいて 入力ミスの可能性をチェックします。

All processing is performed locally within the user's browser. The extension does not collect, store, or transmit any personal data to external servers.

すべての処理はユーザーのブラウザ内で完結します。 本拡張機能は、個人データを外部サーバーへ収集、保存、送信することはありません。

File Download

After the check is completed, the extension generates a result file and allows the user to download it locally. The generated file is created entirely within the browser and is not shared with any third party.

Data Sharing

This extension does not sell, share, or transfer user data to third parties.

External Communication

The extension does not send any data to external servers and does not communicate with any remote services.
